# react-native-turbo-webview
React Native support for turbo framework webviews
## Installation

```sh
npm install react-native-turbo-webview
```

## Usage

```js
import { TurboWebviewView } from "react-native-turbo-webview";

// ...

<TurboWebviewView color="tomato" />
```
